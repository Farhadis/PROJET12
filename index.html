Presentation de deux Projets

Projet OhMyFood !

Ohmyfood est une entreprise de commande de repas en ligne. Elle permet aux utilisateurs de composer leur propre menu et de réduire leur temps d’attente dans les restaurants, car leur menu est préparé à l’avance. Plus de perte de temps à consulter la carte!

Pour realiser ce site j'ai utilisé HTML5 et CSS3. Le développement se faire en CSS, sans JavaScript. Le site est réalisé en adoptant le Mobile First, c’est-à-dire qu’il est d’abord réalisé l'intégration de la maquette mobile, puis tablette, et enfin l'intégration du responsive vers le desktop. Aucun framework ne est utilisé . Tout le code est versionné sur GitHub et il est accessible sur GitHub Pages.

Projet Nina Carducci, Photographe !

Le but de ce projet est de optimiser les images du site d'une photographe : car des images trop lourdes rendent la page très longue à charger. En plus, optimiser le code et sa structure afin d’améliorer le chargement et l’accessibilité du site.

Pour atteindre ce but, les modification suivant sont realisées: Optimisation globale du site tant sur les performances que sur le SEO. Mettre en place le référencement local en utilisant Schema.org. Ajouter les metas pour les réseaux sociaux. Faire les modifications liées à l’accessibilité du site. Produire un rapport d’optimisation présentant toutes nos actions et leur impact. Pour préparer le document détaillant la liste des modifications effectuées, j'ai utilisé le modèle de rapport d’optimisation furnie.

REACT

React est une bibliothèque JavaScript open-source utilisée pour construire des interfaces utilisateur spécifiquement pour des applications d’une seule page. Elle est utilisée pour gérer la couche d’affichage des applications web et mobiles. React est conçu pour permettre de combiner facilement des composants écrits par des personnes, des équipes et des organisations indépendantes.
Les vues déclaratives rendent votre code plus prévisible et plus facile à déboguer. Grâce à React, il est facile de créer des interfaces utilisateurs interactives. Définissez des vues simples pour chaque état de votre application, et lorsque vos données changeront, React mettra à jour, de façon optimale, juste les composants qui en auront besoin.

REDUX

Redux est une bibliothèque JavaScript qui permet de gérer et de mettre à jour l’état d’une application, en utilisant des événements appelés “actions”. Il sert de magasin centralisé pour l’état qui doit être utilisé dans toute l’application, avec des règles garantissant que l’état ne peut être modifié que de manière prévisible. Redux fonctionne avec n’importe quelle couche d’interface utilisateur, et dispose d’un large écosystème d’extensions pour répondre à vos besoins.
Redux est particulièrement adapté aux applications d’une seule page construites avec React, car il facilite la communication entre les composants et la synchronisation de l’état avec l’affichage.
